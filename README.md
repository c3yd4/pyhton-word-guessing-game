"# python-word-guessing-game
This project is a simple yet fun Word Guessing Gme written in Python.
The player tries to guess a randomly selected word within a limited numbers of attempts.

Features
-Randomly selects a word from a predefined word bank

-Player has 10 attempts

-Validates that the input is a single alphabetic character

-Prevents repeated guesses

-Reveals all occurrences of correctly guessed letters

-Displays win message when the full word is guessed

-Shows the correct word if the player runs out of attempts
